# Azure DevOps

**Azure DevOps** is a cloud platform from the Azure Cloud ecosystem that provides a set of tools for the lifecycle of an Application covering development, tests and delivery of the software using the DevOps practices. 

Its a SaaS service hosted by microsoft that acts as an end-to-end ALM/DevOps suite that centralizes: 
* **Repositories**: git based repositories that brings source and version control with branching, pull requests, code reviews and policies. 
* **Project Management**: through board functionality in each project you can handle a Kanban board for the project development. 
* **Azure Pipelines**: bring CI/CD functionalities for Continuous Integration (Compilation, analysis and more) and Continuous Delivery (Deploy) in YAMl or Classic format for building, testing and deploying the applications to Azure, other cloud and even On-Premise servers. 
* **Package management**: in Azure Artifact within each project you can store the application dependencies (NuGet, NPM packages, Java and more).
