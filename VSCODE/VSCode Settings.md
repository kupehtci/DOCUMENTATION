# <span style="color:DodgerBlue">VSCode</span> Settings

VSCode can be configured in almost every aspect of the editor appearance, behaviour and functionality, apart from installing extensions. 

Setting in vscode can be applied at two different levels: 
* **User settings**: applied worldwide to every instance of VS code opened. 
	* Used by default
* **Workspace settings**: stored within a project folder and are only applied when that workspace or folder is open. 
	* Enables project-specific configurations. 

Also the settings can be configured in different 

The new settings in JSON format need to go under `settings.json` file that can be found in: 
* Windows: `%APPDATA%\Code\User\settings.json`
* Linux: `$HOME/.config/Code/User/settings.json`
* MacOS:  `$HOME/Library/Application Support/Code/User/settings.json`
