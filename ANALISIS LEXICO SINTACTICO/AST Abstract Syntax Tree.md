### Abstract Syntax Tree or AST

Its a representation of the Syntax structure of a source code into a tree graph. 

For example, this is the <span style="color:orange;">AST</span> for the following code: 

```PYTHON
while b ≠ 0: 
	if a > b: 
		a := a − b
	else: 
		b := b − a
return a
```
