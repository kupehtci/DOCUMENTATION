# DLM - Amazon Data Lifecycle Manager

**Amazon Data Lifecycle Manager** or **DLM** its an AWS service to automate management tasks of EBS[^1] snapshots and EBS-backed AMIs[^2] using lifecycle policies.

It mainly allows users to define policies that schedule the creation, retention and deletion of EBS snapshots and AMIs. 


[^1]: Elastic Block Storage [[AWS - EBS Elastic Block Store]]
[^2]: Amazon Machine Images or AMIs
