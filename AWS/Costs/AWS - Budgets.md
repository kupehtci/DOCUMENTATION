# AWS - Budgets

**AWS budgets**  is a service for planning, track and control the AWS costas and usage. 

The key features of AWS Budgets are: 

* **Costs Budgets**: you can set predefined costs limits and track AWS spending. 
* **Usage budgets** you can set predefined usage of AWS resources like EC2 hours, storage or Lambda invocations. 
* **Reservation budgets**: monitor reserved instances usage or savings plants bought [^1]. 
* **Alerts**: receive notifications via email or Amazon SNS when the cost exceeds limits or thresholds. 
* **Granular tracking**: budgets can be filtered by service, account, region, tag or usage type. 







[^1]: EC2 and types of EC2. Savings plan and EC2 reserved type. [[AWS - EC2]]
