#AWS 

**AWS Certificate Manager** or **ACM** its a managed service that can: 

* Provision SSL/TLS certificates
* Manage the lifecycle of certificates
* Renew certificates automatically
* Deploy certificates on AWS Services like ELB, CloudFront, API Gateway and others. 


# Integrations

It can be integrated with the following services: 
* **EventBridge**: ACM can send events under certificate approaching expiration, certificate expired and certificate renewed to an EventBridge to trigger other actions.
