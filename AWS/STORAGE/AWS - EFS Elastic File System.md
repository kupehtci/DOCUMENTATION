# AWS - EFS Elastic File System

**AWS EFS** Elastic File System or EFS is a managed network file system in the AWS cloud. 
It provides an serverless and shared file system for linux EC2 instances[^1]

The storage is **serverless** and **elastic**: capacity grows and shrinks automatically as files are added or removed, so there is no need to pre-provision size



[^1]: EC2 or Elastic Cloud Compute are VM that can be deployed in AWS cloud. [[AWS - EC2]]
