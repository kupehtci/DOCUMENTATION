# AWS - Step Functions

**AWS Step Functions** is a fully managed orchestration service to build and automatize workflows that coordinates multiples steps that involves AWS services. 

Can be built with custom code to design an Visual State Machine with built in retries, error handling and parallelization.

Its meant for: 
* **Workflows (Like state machines)** with Tasks, choices to change the workflow depending on an input or task result, Parallel, Mappings, Wait adn PAss to define the order, branches and flow of the process. 
* Executions are individual runs of an Step Function where it tracks its progress, inputs or outputs and logs every step for audit. 

It integrates with most AWS Services like Lambda, Fargate, DynamoDB, SQS / SNS, Batch, Glue and more.
