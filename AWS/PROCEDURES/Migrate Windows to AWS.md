# Migrate Windows to AWS

To migrate windows's data from on-premises to AWS Cloud without disrumpting workloads: 

1. Deploy and configure Amazon FSx for Windows File Server on AWS. 
2. Deploy and configure an Amazon FSx File Gateway on premises. 
3. Move the on-premises file data to the FSx File Gateway. 
4. Configure the cloud workloads to use FSx for Windows File Server on AWS. 
5. Configure the on-premises workloads to use the FSx File Gateway.
