# Amazon Pinpoint journey

**Amazon Pinpoint journey** is an AWS service that allow to configure a set of steps to communicate with end users depending on their characteristics or usage. 

A **journey** defines the group of users that will participate in the communication channel. Each jurney is composed about a set of activities like: 
* Sending an Email
* SMS
* Push notificacion
* Sending users into different paths based on their behaviors. 

You can interact with them under events (Like a welcome email) and respond to back-events like send a follow-up message for those who didn't open the email.
