# Pandas

**Pandas** is a python library for data analysis that provide complex data structures like Series and DataFrames. 

It makes easier to manipulate, clean and analyze tabulated data. 

Pandas main characteristics are:

* Series and DataFrames as main data structures
* Data manipulation for filtering, aggregation, fusion and more. 
* Manipulation of remaining data, allowing to handle NaN values. 
* Operations over temporal series of data: functionalities specialized in time-related data. 
* Read and write of data: allowing to extract data from CSV, Excel, SQL and more.   

Also its commonly used because: 
* **Efficiency**: can process fastly a high volume of data. 
* **Versatility**: compatible with multiple sources of data and formats
* **Active**: used by the community with lots of documentation. 
* **Integration**: can be easily integrated with other libraries like NumPY, Matplotlib and Scikit-learn. 


To use pandas, import it as a library once it has been installed: 
```python
import pandas as pd
```
