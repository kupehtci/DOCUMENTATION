# PYTHON - Print()

`print(...)` function can be used in python to prompt text into the console where the script has been called, like: 
```python
print("Hello world!")
```

This function can also be used to prompt variable's value like: 
```python
name = "Daniel"
# single variable
print(name)

# Concatenating
print("Hello " + name)

# using coma
print("Hello ", name)

# using f
print(f"Hello {name}")
