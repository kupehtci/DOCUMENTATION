### Azure service principal

An <span style="color:DodgerBlue;">Azure Service Principal</span> is a definition of policies and permissions that has an application within Azure. 

Its a main element of the RBAC[^1] within Azure. 


[^1]: Role Base Access Control is the control of permissions based in granting roles with allows and deny permissions to certain resources within Azure. [[AZURE - RBAC]]
