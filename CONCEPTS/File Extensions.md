# File Extensions


# Documents and office related files


| Extension                | Meaning                                        | Purpose                                                                      |
| :----------------------- | ---------------------------------------------- | ---------------------------------------------------------------------------- |
| `.txt`                   | Text File                                      | Plain Text file                                                              |
| `.md`                    | Markdown file                                  | Plain Text markdown format file                                              |
| `.doc` or `.docx`        | Microsoft Word                                 | Word file                                                                    |
| `.xls` or `.xlsx`        | Microsoft Excel                                | Excel file                                                                   |
| `.csv`                   | Comma sepparated Value                         | Column-based file with plain text separated by commas. Can be read by Excel. |
| `.pdf`                   | Portable Document Format                       | PDF Format                                                                   |
| `.odt`, `.ods` or `.odp` | OpenDocument Text, Spreadsheet or Presentation | OpenDocument format files                                                    |
| `.rtf`                   | Rich Text Format                               |                                                                              |

# Security related files extensions

| Extension | Meaning                     | Purpose                                  |
| :-------- | --------------------------- | ---------------------------------------- |
| `.pem`    | Private Enhanded Mail       | Used for certficates and Key Pairs       |
| `.cer`    | Certificate                 | Certificate DER or PEM encoded           |
| `.crt`    | Certificate                 | Not encoded                              |
| `.key`    | Private Key File            | Unecrypted or encrypted                  |
| `.csr`    | Certificate Signing Request | CA exchange of certificates              |
| `.asc`    | ASCII armored               | OpenPGP and GPG keys and encrypted files |
| `.pub`    | Public Key                  | Public key of a key pair.                |
