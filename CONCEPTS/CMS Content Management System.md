# CMS Content Management System

A **CMS** or **Content Management System** is a software that allow users to create, manage and modify digital content without requiring technical knowledge like web programming. 

It needs to provide a user friendly interface for adding content, designing and managing web content. 


There are some commonly used CMS platforms: 
* WordPress
* Drupal
* Wix
* Magento or Adobe Commerce
* Shopify
* SalesForce
