# API

**API** or **Application Programming Interface** its a contract that define the quests data models and actions that can be done over an application through rules, formats or endpoints. 

Its like a set of rules or protocols to use to enable communication between two different software parties. 

Common types are: 

* **API REST** [^1]: API that follows REST architectural design where the resource-oriented URLs are stateless (Don't store status or data) and have cacheable interactions or requests.
	* **HTTP / JSON APIs**: JSON based API RESTful APIs
* **WebSocket APIs** (Bidirectional): Websocket protocol provides a full-duplex connection between client and server so both can exchange messages. 
* **SOAP (XML-based)**: Messaging protocol for web that uses XML to send requests and responses. 


[^1]: API RESTful [[API REST - HTTP Methods]]
