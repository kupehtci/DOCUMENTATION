# WebSocket

An WebSocket API[^1] is a network protocol that provides a full-duplex (bi-directional) communication endpoint where both parties can exchange messages. 

It establishes a long-lived TCP connection between a client and a server to send messages at any time, begining with an HTTP/1.1 handshake between both parties to Upgrade the connection. 






[^1]: API or Application Programming Interface [[API - Application Programming Interface]]
