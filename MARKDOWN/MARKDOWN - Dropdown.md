#MARKDOWN 

In order to generate a dropdown that can contain text that is not visible until you dropdown it: 


```MARKDOWN
<details>
<summary>HEADING 1</summary>

This is a bulet list

+ <details>
	<summary>HEADING 1.1</summary>

	Hello from inside dropdown
	</details>

</details>

```
